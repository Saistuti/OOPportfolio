(3 marks)


a. What is the latest version of the Java SDK that is available?\
JDK 25.0.2 is the latest version of Java SDK that is available.


b. What is the difference between Java SE and Java ME?\
Java SE (Standard Edition) is used to develop desktop and large-scale applications and provides complete libraries and tools. Java ME (Micro Edition) is designed for small or embedded devices with limited memory and resources. In short, Java SE is for computers, while Java ME is for smaller devices.


c. Which operating system is Java available for?\
Java is platform-independent and can run on different operating systems such as Windows, macOS, Linux, and Unix, as long as the Java Virtual Machine (JVM) is installed.


d. Give some other IDEs for Java apart from Eclipse nd IntelliJ?\
Other IDEs for Java include NetBeans, BlueJ, DrJava, JDeveloper, and Android Studio.


e. What is the main() function for in a Java program?
The main() function is the starting point of a Java program. The program execution begins from this method when it runs.
