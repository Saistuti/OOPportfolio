**1 - Sphere
Design and implement a class called Sphere that contains instance data that represent a sphere’s diameter
-Define a Sphere constructor to accept and initialize the diameter.
-Include getter and setter methods for the diameter.
-Include methods to calculate and return the volume and surface area of the sphere.
-Include a toString method that returns a one-line description of the sphere.
-Create a driver class (main class) called MultiSphere, whose main method instantiates and updates several Sphere objects.
(3 marks)**

// Class Sphere
```
class Sphere {
private double diameter;

    // Constructor
    public Sphere(double diameter) {
        this.diameter = diameter;
    }

    // Getter
    public double getDiameter() {
        return diameter;
    }

    // Setter
    public void setDiameter(double diameter) {
        this.diameter = diameter;
    }

    // Method to calculate radius
    private double getRadius() {
        return diameter / 2;
    }

    // Method to calculate volume
    public double getVolume() {
        double r = getRadius();
        return (4.0 / 3.0) * Math.PI * Math.pow(r, 3);
    }

    // Method to calculate surface area
    public double getSurfaceArea() {
        double r = getRadius();
        return 4 * Math.PI * Math.pow(r, 2);
    }

    // toString method
    public String toString() {
        return "Sphere [Diameter = " + diameter +
                ", Volume = " + String.format("%.4f", getVolume()) +
                ", Surface Area = " + String.format("%.4f", getSurfaceArea()) + "]";
    }
}
```

```
}
```

**3 - Bulb
Design and implement a class called Bulb that represents a light bulb that can be turned on and off.
-Create a driver class called Lights, whose main method instantiates and turns on some Bulb objects
(3 marks)**

// Class Bulb
```
class Bulb {
private boolean isOn;

    // Constructor
    public Bulb() {
        isOn = false;
    }

    // Method to turn ON
    public void turnOn() {
        isOn = true;
    }

    // Method to turn OFF
    public void turnOff() {
        isOn = false;
    }

    // Method to check status
    public boolean isOn() {
        return isOn;
    }

    public String toString() {
        if (isOn)
            return "Bulb is ON";
        else
            return "Bulb is OFF";
    }
}

```

// Main class Lights
```
public class Lights {
```

```
class Account {
private String name;
private int accountNumber;
private double balance;

    // Constructor with full details
    public Account(String name, int accountNumber, double balance) {
        this.name = name;
        this.accountNumber = accountNumber;
        this.balance = balance;
    }

    // New Constructor
    public Account(String name, int accountNumber) {
        this.name = name;
        this.accountNumber = accountNumber;
        this.balance = 0.0;
    }

    public void deposit(double amount) {
        balance += amount;
    }

    public void withdraw(double amount) {
        balance -= amount;
    }

    public String toString() {

                "\nBalance: " + balance;
    }
}
```

**5 - Cards
Design and implement a class called Card that represents a standard playing card. Each
card has a suit and a face value. Create a program that deals five random cards.
(3 marks)**

// Class Card
```
import java.util.Random;

class Card {
private String suit;
private String face;

    // Arrays for suits and face values
    private static final String[] suits = {"Hearts", "Diamonds", "Clubs", "Spades"};
    private static final String[] faces = {
            "2", "3", "4", "5", "6", "7", "8", "9", "10",
            "Jack", "Queen", "King", "Ace"
    };

    // Constructor (random card)
    public Card() {
        Random rand = new Random();
        suit = suits[rand.nextInt(suits.length)];
        face = faces[rand.nextInt(faces.length)];
    }

    // toString method
    public String toString() {
        return face + " of " + suit;
    }
}
```
