**2 - Write a program that simulates rolling two dice using the following steps:
1.Prompt the user for the number of sides for two dice.
2. “Roll” the dice three times by generating a random number between 1 (inclusive) and the number of sides (inclusive).
3. Keep track of the sum of the rolls for each die and output the sum and average for each die.
(3 marks)**


Sample Output:


How many sides does die 1 have? 6

How many sides does die 2 have? 20

Die 1 first roll = 5.

Die 2 first roll = 14.

Die 1 second roll = 1.

Die 2 second roll = 20.

Die 1 third roll = 3.

Die 2 third roll = 9.

Die 1 rolled a total of 9 and rolled 3 on average.

Die 2 rolled a total of 43 and rolled 14.333 on average.


```
import java.util.Scanner;
import java.util.Random;

public class Dice {
    public static void main(String[] args) {

        Scanner input = new Scanner(System.in);
        Random rand = new Random();

        // Step 1: Ask for number of sides
        System.out.print("How many sides does dice 1 have? ");
        int sides1 = input.nextInt();

        System.out.print("How many sides does dice 2 have? ");
        int sides2 = input.nextInt();

        int sum1 = 0;
        int sum2 = 0;

        // Step 2: Roll dice three times
        for (int i = 1; i <= 3; i++) {

            int roll1 = rand.nextInt(sides1) + 1;
            int roll2 = rand.nextInt(sides2) + 1;

            sum1 += roll1;
            sum2 += roll2;

            if (i == 1) {
                System.out.println("Dice 1 first roll = " + roll1 + ".");
                System.out.println("Dice 2 first roll = " + roll2 + ".");
            }
            else if (i == 2) {
                System.out.println("Dice 1 second roll = " + roll1 + ".");
                System.out.println("Dice 2 second roll = " + roll2 + ".");
            }
            else {
                System.out.println("Dice 1 third roll = " + roll1 + ".");
                System.out.println("Dice 2 third roll = " + roll2 + ".");
            }
        }

        // Step 3: Calculate averages
        double avg1 = sum1 / 3.0;
        double avg2 = sum2 / 3.0;

        // Output results
        System.out.println("Dice 1 rolled a total of " + sum1 + " and rolled " + avg1 + " on average.");
        System.out.println("Dice 2 rolled a total of " + sum2 + " and rolled " + avg2 + " on average.");

        input.close();
    }
}
```

**3 - Write a block of source code to encrypt a four digit pin number by doing the
following:
1.Convert the pin number to hexadecimal.
2. Generate two random numbers greater than 1000 and less than 65536 and convert them
   to hexadecimal.
3. Sandwich the converted pin between the two random hexadecimal numbers. (3 marks)**

Sample Output:

Enter a 4 digit pin number to encrypt: 8491

Your encrypted pin number is 1787212b12ef.


```
import java.util.Scanner;
import java.util.Random;

public class PinEncryption {
    public static void main(String[] args) {

        Scanner input = new Scanner(System.in);
        Random rand = new Random();

        // Ask user for PIN
        System.out.print("Enter a 4 digit pin number to encrypt: ");
        int pin = input.nextInt();

        // 1. Convert PIN to hexadecimal
        String pinHex = Integer.toHexString(pin);

        // 2. Generate two random numbers between 1000 and 65535
        int rand1 = rand.nextInt(65536 - 1000) + 1000;
        int rand2 = rand.nextInt(65536 - 1000) + 1000;

        // Convert them to hexadecimal
        String hex1 = Integer.toHexString(rand1);
        String hex2 = Integer.toHexString(rand2);

        // 3. Sandwich the PIN hex between the two random hex numbers
        String encryptedPin = hex1 + pinHex + hex2;

        // Display result
        System.out.println("Your encrypted pin number is " + encryptedPin + ".");

        input.close();
    }
}
```
---
Enter a 4 digit pin number to encrypt: 5093

Your encrypted pin number is f1ec13e5c15.

**4 - Write and application that reads the radius of a sphere and prints its volume and
surface area. Use the following formulas, where r represents the radius of the sphere.
Print the output to four decimal places.
Volume = 4πr3 / 3
Surface area = 4πr2
(3 marks)**

```
import java.util.Scanner;

public class SphereCalculation {
    public static void main(String[] args) {

        Scanner input = new Scanner(System.in);

        // Ask user for radius
        System.out.print("Enter the radius of the sphere: ");
        double r = input.nextDouble();

        // Calculate volume and surface area
        double volume = (4 * Math.PI * Math.pow(r, 3)) / 3;
        double surfaceArea = 4 * Math.PI * Math.pow(r, 2);

        // Print results to four decimal places
        System.out.printf("Volume of the sphere = %.4f\n", volume);
        System.out.printf("Surface area of the sphere = %.4f\n", surfaceArea);

        input.close();
    }
}
```
---
Enter the radius of the sphere: 7

Volume of the sphere = 1436.7550

Surface area of the sphere = 615.7522
