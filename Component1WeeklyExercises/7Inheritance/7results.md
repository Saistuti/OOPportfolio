**1 - Hospital
Design and implement a set of classes that define the employees of a hospital: doctor,
nurse, receptionist, cleaner etc. Include methods in each class that are named
according to the services provided by that person and that print an appropriate
message.
Eg Nurse <name> Emp# has <x> patients
Doctor <name> Emp# specializes in <specialism>
Cleaner <name> Emp# of <department> is sweeping
Create a driver class to instantiate and test the classes
(6 marks)**

// Employee class 
```
class Employee {
String name;
int empId;

    Employee(String name, int empId) {
        this.name = name;
        this.empId = empId;
    }

    void displayInfo() {
        System.out.println("Employee: " + name + ", ID: " + empId);
    }
}
```

**2 - Player Statistics
Design and implement a set of classes that keep track of various player sports
statistics (PlayerStats). Have a lower level (sub) class that represent a specific
sport (FootballStats, CricketStats, CyclingStats etc...). Tailor the services of the
classes to the sport in question, with common attributes and methods in the 
higher-level class, as appropriate. Create a driver class to instantiate and text instances
of the classes.
(6 marks)**


// PlayerStats class
```
abstract class PlayerStats {
String name;
int age;
int matchesPlayed;

    PlayerStats(String name, int age, int matchesPlayed) {
        this.name = name;
        this.age = age;
        this.matchesPlayed = matchesPlayed;
    }

    void displayBasicInfo() {
        System.out.println("Player: " + name +
                ", Age: " + age +
                ", Matches: " + matchesPlayed);
    }

    // Abstract method (must be implemented by subclasses)
    abstract void displayStats();
}
```
// FootballStats subclass
```
class FootballStats extends PlayerStats {
int goals;
int assists;

    FootballStats(String name, int age, int matchesPlayed, int goals, int assists) {
        super(name, age, matchesPlayed);
        this.goals = goals;
        this.assists = assists;
    }

    void displayStats() {
        System.out.println("Football Player " + name +
                " scored " + goals + " goals and made " +
                assists + " assists");
    }
}
```
